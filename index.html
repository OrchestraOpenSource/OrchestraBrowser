    About Orchestra Browser

    An Internet Browser that follow the web standards). We aim to make a browser that just works. If a website does not works on Orchestra Browser, it does not follows the web standards).

    Why use Orchestra Browser?

    1. Don't use it right now on production! This browser is just at the beginning of it's life and it's very experimental.
    2. It's fast.
    3. It's good looking.
    4. It's the closest you will ever get to the perfect web surfing.

    About our view of Open Source.

    We Believe in Open Source Code.

    The future is Open, you must decide where you want to go, what you want to do, how and when. If you don't like something, then you must be able to change it. And you don't have to! The truly Open Source philosophy says:

    • You can read the code and see how it was made, but you don't have to.
    • You can change the code and adapt it to your needs, or even fork it and start your own thing based on that code, but you don't have to.
    • You can and should contribute back to the original author with your code improvements, but you don't have to.
